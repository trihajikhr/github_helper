# Creating an account on GitHub


## Tentang Akun Pribadi Anda di GitHub

Untuk memulai dengan GitHub, Anda harus membuat akun pribadi gratis dan memverifikasi alamat email Anda.

Anda juga dapat melakukan otentikasi dengan Google atau Apple — yang merupakan penyedia *social login* (masuk sosial) yang didukung saat Anda membuat akun di GitHub. Untuk pengguna iOS, meskipun Anda telah mengaktifkan pengaturan "Sembunyikan Alamat Email Saya (*Hide My Email addresses*)" untuk akun Apple Anda, menggunakan *social login* akan menghasilkan pembuatan akun GitHub baru.

Setiap orang yang menggunakan GitHub masuk ke akun pengguna (*user account*). Akun pengguna Anda adalah identitas Anda di GitHub dan memiliki nama pengguna (*username*) serta profil. Sebagai contoh, lihat profil [@octocat's profile](https://github.com/octocat).

Nantinya, Anda dapat menjelajahi berbagai jenis akun yang ditawarkan GitHub, dan memutuskan apakah Anda memerlukan paket penagihan (*billing plan*). Untuk informasi lebih lanjut, lihat [Types of GitHub accounts](https://docs.github.com/en/get-started/learning-about-github/types-of-github-accounts) dan  [GitHub’s plans](https://docs.github.com/en/get-started/learning-about-github/githubs-plans).

> Perlu diperhatikan bahwa langkah-langkah dalam artikel ini tidak berlaku untuk Pengguna yang Dikelola Perusahaan (*Enterprise Managed Users*). Jika akun GitHub Anda telah dibuatkan oleh perusahaan Anda, Anda dapat melewati artikel ini dan melanjutkan ke [Hello World](https://docs.github.com/en/get-started/start-your-journey/hello-world).

## Mendaftar Akun Pribadi Baru

1.  Arahkan ke [GitHub · Change is constant. GitHub keeps you ahead. · GitHub](https://github.com/.)
2.  Klik **Sign up** (Daftar).
3.  Sebagai alternatif, klik **Continue with Google** (Lanjutkan dengan Google) untuk mendaftar menggunakan *social login*.
4.  Ikuti petunjuk untuk membuat akun pribadi Anda.

**Penting:** Selama pendaftaran, Anda akan diminta untuk memverifikasi alamat email Anda. Tanpa alamat email yang terverifikasi, Anda tidak akan dapat menyelesaikan beberapa tugas dasar GitHub, seperti membuat repositori.

> **Catatan Tambahan:** Beberapa perusahaan membuat akun pengguna yang dikelola (*managed user accounts*) untuk penggunanya. Anda tidak dapat mendaftar untuk akun pribadi dengan alamat email yang sudah terverifikasi untuk akun pengguna yang dikelola.
>
> Jika Anda mengalami masalah dalam memverifikasi alamat email Anda, ada beberapa langkah pemecahan masalah yang dapat Anda ambil. Untuk informasi selengkapnya, lihat [Verifying your email address](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-email-preferences/verifying-your-email-address#troubleshooting-email-verification).

## Langkah Selanjutnya

Setelah Anda berhasil membuat akun pribadi Anda, kita akan mulai menjelajahi dasar-dasar GitHub. Dalam tutorial berikutnya, [Hello World](https://docs.github.com/en/get-started/start-your-journey/hello-world), Anda akan mempelajari tentang repositori dan cara membuatnya, serta akan diperkenalkan dengan konsep-konsep seperti *branching* (pencabangan), *commits* (komit), dan *pull requests* (permintaan tarik).

Kami sangat menyarankan Anda untuk mengonfigurasi 2FA (Otentikasi Dua Faktor) untuk akun Anda. 2FA adalah lapisan keamanan ekstra yang dapat membantu menjaga akun Anda tetap aman. Untuk informasi lebih lanjut, lihat [Configuring two-factor authentication](https://docs.github.com/en/authentication/securing-your-account-with-two-factor-authentication-2fa/configuring-two-factor-authentication).
